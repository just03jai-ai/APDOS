# FarMart Repo Map

Use this map to choose candidate repos from a parent workspace that contains
multiple FarMart repos. Verify with code search before editing.

## How To Use This Map

1. Match the user's words to the **first hints** column.
2. If the user explicitly names a repo or gives a strict repo instruction, use
   that repo first. Otherwise search the likely repo first; only expand to
   connected repos when code evidence points there.

## Repository Entry Docs

When working inside a repo, fully read these files if they are available:

- `README.md` or `Readme.md`
- `AGENTS.md`
- `CLAUDE.md`

If the selected repo has a nested app, package, or service that clearly owns the
change, also read the same entry docs from that nested directory when available.
Then load any companion skill named by the repo map before editing.
